一个包可以在源文件中多次导入

Go 源文件可以多次导入相同的包, 但导入名称必须不同. 这些相同的包导入引用相同的包实例.

例如:

package mypkg 后的注释 // import “x.y.z.mypkg” 对于 Go 编译器是有意义的

例如, 当导入此包的源文件由标准 Go 编译器编译时, 以下包的导入路径必须为 “x.y.z.mypkg”.
